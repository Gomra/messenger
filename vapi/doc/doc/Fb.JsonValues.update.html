update

Description:

public bool update () throws Error

Updates the current working root.

This should be called after all of the FbJsonValue's have been added with add(). If an array was set with set_array(), then this should be called in a while loop, until FALSE is returned.

Returns:

TRUE if the values were updated, otherwise FALSE.

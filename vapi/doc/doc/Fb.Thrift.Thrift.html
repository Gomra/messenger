Thrift

Description:

public Thrift (ByteArray bytes, uint offset)

Creates a new Thrift.

The returned Thrift should be freed with unref() when no longer needed. This will optionally use a ByteArray at an offset, rather than a newly created and internal ByteArray.

Parameters:

bytes

The ByteArray to read or write.
offset

The offset in bytes of the data in bytes.

Returns:

The new Thrift.

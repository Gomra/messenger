Description:

public ApiThread dup (bool deep)

Duplicates an ApiThread.

If thrd is NULL, a new zero filled ApiThread is returned. The returned ApiThread should be freed with free () when no longer needed.

Parameters:

deep

TRUE to duplicate allocated data, otherwise FALSE.

Returns:

The new ApiThread.

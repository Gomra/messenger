ORCA_AGENT

Description:

public const unowned string ORCA_AGENT

The part of the user agent that looks like the official client, since the server started checking this.

We announce ourselves as compatible with Orca-Android 38.0 since that's the closest version to the last major protocol update. Some parts use older features, some parts use newer ones.

Fun fact: this version sends old-style MQIsdp CONNECT messages for the first connection, with JSON payloads instead of compressed thrift.

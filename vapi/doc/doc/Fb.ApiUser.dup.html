dup

Description:

public ApiUser dup (bool deep)

Duplicates an ApiUser.

If user is NULL, a new zero filled ApiUser is returned. The returned ApiUser should be freed with free() when no longer needed.

Parameters:

deep

TRUE to duplicate allocated data, otherwise FALSE.

Returns:

The new ApiUser.

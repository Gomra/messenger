public enum MqttConnectFlags

The MqttMessage flags for the CONNECT message.


Namespace: Fb
Package: doc

Content:

Enum values:

  • CLR - Clear the session.
  • WILL - A will message is in the payload.
  • RET - Retain the will message.
  • PASS - A password is in the payload.
  • USER - A user name is in the payload.
  • QOS0 - Use no quality of service.
  • QOS1 - Use level one quality of service.
  • QOS2 - Use level two quality of service.
